# Spanish Verb Wheel

An interactive Spanish verb conjugation trainer for students.

Students shuffle a verb, tense, and subject, then type the matching conjugation. The app gives immediate feedback and keeps a local mistake history for review.

## Privacy

This is a static browser app. Typed answers, scores, and mistake history are stored only in the user's browser with `localStorage`. There is no login, no microphone access, and no practice data is sent to a server.
